A little project description for enterprise social network
-----------------------------------------------------------------

Social Network is a direction of the enterprise management practise. It will impact the 
enterprise communication and management power distribution.This is a demo project
to catch the trend and common practice in the enterprise marketing social network on the public
social network.

Project Structure Introduce
--------------------------------------------------------------
Design			the directory to store the psd of photoshop of the system
Prototype		the directory to store the html/js based online demo system
Doc				the directory to store the document of the system
